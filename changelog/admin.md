# Changelog — Панель администратора

Все изменения в функциях и интерфейсе панели администратора.

---

## [Этап 1] 2026-05-06 — 2026-05-18

### Добавлено
- Панель администратора — начальная mock-версия (F-10, F-11)
- Бонусная система — mock (F-12)
- GPS-отслеживание курьера — начальная версия (F-13)

---

## 2026-05-31 — 2026-06-05

### Добавлено
- Admin API — заказы: `GET /api/admin/orders`, `PATCH /api/admin/orders/:id` (изменение статуса)
- Admin API — меню: `GET /api/admin/menu`, `POST /api/admin/menu`, `PATCH /api/admin/menu/:id`
- Admin API — бонусы: управление активными акциями
- Admin API — SMS-рассылка: отправка акций клиентской базе через Twilio
- Панель администратора подключена к реальному API (вместо mock-данных)

---

## 2026-06-06 — 2026-06-07

### Добавлено
- Страница входа `/admin/login` — форма email + пароль, проверка роли `admin` через Supabase

---

## 2026-06-18 — 2026-06-19

### Добавлено
- `POST /api/admin/orders` — ручное создание заказов по телефону или на месте (без онлайн-оплаты)
- Скрытие заказов в панели в течение 3-минутного окна отмены — администратор не видит заказ, пока клиент ещё может его отменить, чтобы не начать готовить преждевременно
- **F-14:** кнопка паузы приёма заказов — администратор может временно приостановить приём новых заказов
