# Changelog — Frontend

Все изменения в интерфейсе: страницы, компоненты, дизайн, анимации.

---

## [Этап 1] 2026-05-06 — 2026-05-18

### Добавлено
- Инициализация проекта — Next.js 15 App Router + TypeScript
- Дизайн-система: цвета, типографика, базовые компоненты
- `Header` с навигацией и разделом «Отзывы»
- `HeroSection` — главный экран с заголовком и CTA
- Корзина (`CartContext`) — добавление, удаление, счётчик позиций
- Страница оформления заказа (`/commande`) — форма адреса и доставки
- Страница авторизации клиента (`/connexion`)
- Страница отслеживания заказа (`/order-tracking`)
- Страница оплаты Stripe (`/paiement`) — начальная mock-версия
- Панель администратора (`/admin`) — начальная mock-версия
- Система i18n — французский / английский / русский с переключением в реальном времени
- `Footer` — адрес, часы работы, навигация
- Секция About
- Секция Reviews

### Исправлено
- Mobile touch events — некорректная обработка на iOS
- Hydration mismatch в компонентах корзины
- i18n для пустого меню — пустой экран вместо сообщения
- Hero-секция занимала весь экран — меню не было видно без прокрутки
- Соцсети перенесены в среднюю колонку footer
- Ссылка «Отзывы» перенесена в header

## 2026-05-31 — 2026-06-05

### Добавлено
- Страница оплаты (`/paiement`) подключена к Stripe Payment Intent (реальный API)
- Страница подтверждения заказа (`/confirmation`) — показывает детали после оплаты
- Страница оформления заказа (`/commande`) подключена к реальному API
- Меню подключено к реальному API (`GET /api/menu`)
- Бонус подключён к реальному API
- Поле комментария к заказу на странице оформления
- Страница истории заказов (`/account`) — список прошлых заказов
- Кнопка «Повторить заказ» — копирует состав прошлого заказа в корзину
- Шум-текстура на фон — эффект матового стекла/материала
