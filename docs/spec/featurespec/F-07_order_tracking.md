# F-07 — История заказов (для зарегистрированных клиентов)

**Этап:** Backend

**Назначение:** Зарегистрированный клиент может видеть список своих прошлых заказов с составом, суммой и статусом.

> Публичная страница отслеживания (`/suivi/`) удалена. Чек об оплате отправляется на email через Stripe (см. F-03, F-08). Анонимные клиенты отслеживать заказ не могут.

---

## Что видит пользователь

- Список всех своих заказов: дата, состав, итоговая сумма, статус
- Статусы: Новый → Готовится → В доставке → Выполнен / Отменён

---

## Действия пользователя

- Заходит в аккаунт
- Переходит в раздел "Мои заказы"
- Видит историю заказов с текущим статусом каждого

**API:** `GET /api/users/me/orders` (требует авторизации — Client JWT)

---

## Исключительные случаи

- Клиент не авторизован → перенаправление на страницу входа
- Заказов ещё не было → пустой список с текстом "У вас пока нет заказов"
- Сервер недоступен → "Не удалось загрузить историю заказов. Попробуйте позже."
