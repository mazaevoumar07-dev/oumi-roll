# F-13 — GPS-отслеживание курьера

**Статус: УДАЛЕНО**

> Функция GPS-отслеживания курьера удалена из проекта.
>
> Причина: публичная страница отслеживания `/suivi/` удалена. Клиент больше не видит карту с курьером — чек об оплате отправляется на email, статус заказа виден только зарегистрированным пользователям в истории заказов (F-07).
>
> Удалено: `courier_token`, таблица `courier_locations`, страница `/courier/[id]`, эндпоинты `POST /api/courier/location` и `GET /api/orders/:id/location`.
>
> Администратор по-прежнему меняет статус заказа на "В доставке" — просто без генерации ссылки для GPS-курьера.
