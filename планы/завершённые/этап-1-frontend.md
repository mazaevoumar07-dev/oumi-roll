# Этап 1 — Frontend

**Статус:** Завершён ✓  
**Дата завершения:** до 2026-05-02

---

## Что было сделано

Разработан полный статичный сайт без сервера и без API. Всё работает на стороне клиента.

### Страницы

| Страница | Путь | Описание |
|---|---|---|
| Главная | `/` | Hero-секция + меню роллов |
| Оформление заказа | `/commande` | Форма с данными клиента и выбором доставки/самовывоза |
| Оплата | `/paiement/[id]` | Страница оплаты (заглушка, без реального Stripe) |
| Отслеживание | `/suivi/[id]` | Страница статуса заказа (заглушка) |
| Вход / Регистрация | `/connexion` | Форма входа и регистрации (заглушка) |
| Панель администратора | `/admin` | Панель управления (заглушка) |

### Компоненты

- Header с логотипом, корзиной, переключателем языка и кнопкой "Войти"
- Footer с контактами, навигацией и соцсетями
- Карточки роллов (фото, название, описание, цена, кнопка "Добавить")
- Корзина (боковая панель или отдельная страница)
- Форма оформления заказа
- Секции: About, Reviews

### Дизайн

- Тёмная премиальная эстетика: фон `#0D0D0D`, акценты `#C8A96E` (золото)
- Шрифты: Cormorant Garamond (заголовки) + DM Sans (текст)
- Mobile-first, адаптив: mobile / tablet / desktop
- Иконки: Lucide Icons

---

## Что НЕ делалось на этом этапе

- Никакого сервера и API
- Данные меню — статичные (хранились в коде)
- Корзина — только в памяти браузера, не сохранялась
- Оплата, регистрация, SMS — заглушки без реальной логики
