# Design Spec — OUMI ROLL

## Общий стиль

**Концепция:** Премиальный японский ресторан. Тёмная, элегантная эстетика с золотыми акцентами. Сайт должен ощущаться как дорогое место — спокойно, минималистично, с достоинством.

**Ключевые слова:** тёмный, золотой, японский, люксовый, минималистичный, изысканный.

## Типографика

### Шрифты

| Роль              | Шрифт                  | Откуда         |
|-------------------|------------------------|----------------|
| Заголовки (H1–H2) | **Cormorant Garamond** | Google Fonts   |
| Подзаголовки (H3–H4) | **Cormorant Garamond** Medium | Google Fonts |
| Основной текст    | **DM Sans**            | Google Fonts   |
| Акцентные подписи / лейблы | **DM Sans** Light | Google Fonts |

**Почему эти шрифты:**
- `Cormorant Garamond` — серифный шрифт с тонкими штрихами, ощущение роскоши и европейского изыска. Идеально для заголовков на тёмном фоне с золотым цветом.
- `DM Sans` — современный гротеск, хорошо читается на тёмном фоне на мобильных экранах. Нейтральный, не отвлекает.

### Размеры (мобиль → десктоп)

| Элемент       | Mobile  | Desktop |
|---------------|---------|---------|
| H1            | 32px    | 56px    |
| H2            | 24px    | 40px    |
| H3            | 20px    | 28px    |
| Body          | 15px    | 16px    |
| Small / Label | 12px    | 13px    |

**Line-height:** 1.6 для тела, 1.2 для заголовков.  
**Letter-spacing:** заголовки +0.04em, лейблы в верхнем регистре +0.12em.

## Логотип

### Концепция

Логотип состоит из двух частей: **иконка** + **текст**.

**Иконка:**
- Стилизованная волна (японский мотив — отсылка к «Большой волне» Хокусая), упрощённая до одной плавной линии.
- Внутри или над волной — тонкий круг, символизирующий ролл в разрезе.
- Исполнение: линейный (outline) стиль, цвет — Gold Primary `#C8A96E`.
- Размер иконки: квадрат ~40×40px в шапке, ~80×80px на загрузочном экране.

**Текст:**
- Две строки:
  - `OUMI` — крупно, шрифт Cormorant Garamond Bold, цвет Off-White `#F0EAD6`
  - `ROLL` — чуть мельче или того же размера, с letter-spacing +0.2em, цвет Gold Primary `#C8A96E`
- Или одна строка: `OUMI ROLL` — где `ROLL` выделен золотым цветом.

**Варианты логотипа:**
| Вариант         | Применение                          |
|-----------------|-------------------------------------|
| Полный (иконка + текст горизонт.) | Шапка десктопа       |
| Компактный (иконка + текст верт.) | Шапка мобайла, футер |
| Только иконка   | Favicon, PWA-иконка, загрузка       |

**Формат файлов:** SVG (основной), PNG 1x/2x/3x.

## Компоненты интерфейса

### Кнопки

| Тип             | Фон         | Текст      | Рамка           | Hover               |
|-----------------|-------------|------------|-----------------|---------------------|
| Primary CTA     | `#C8A96E`   | `#0D0D0D`  | —               | `#E2C07A`, scale 1.02 |
| Secondary       | transparent | `#C8A96E`  | 1px `#C8A96E`   | фон `#C8A96E1A`    |
| Danger          | transparent | `#C0392B`  | 1px `#C0392B`   | фон `#C0392B1A`    |
| Ghost / Link    | —           | `#8A8A8A`  | —               | цвет `#F0EAD6`     |

Радиус скругления: **4px** (не круглые — угловатость подчёркивает строгость стиля).  
Padding: `12px 24px` для стандартных, `10px 16px` для компактных.

### Карточка блюда

```
┌────────────────────────────┐
│  [фото блюда  16:9]        │  ← фото занимает верхние 60%
├────────────────────────────┤
│  Имя блюда   (H3, Off-White│
│  Краткое описание (Muted)  │
│  ─────────────────────     │  ← тонкая линия Gold Subtle
│  1 290 ₽          [+]      │  ← цена Off-White, кнопка Gold
└────────────────────────────┘
```

- Фон карточки: `#1A1A1A`
- Рамка: 1px `#2A2A2A`, при hover — 1px `#C8A96E`
- Transition: 200ms ease

### Поля ввода (Input / Textarea)

- Фон: `#1A1A1A`
- Рамка по умолчанию: 1px `#2A2A2A`
- Рамка в фокусе: 1px `#C8A96E`
- Текст: `#F0EAD6`
- Placeholder: `#8A8A8A`
- Label: `#8A8A8A`, uppercase, letter-spacing 0.1em, 12px

### Навигация (Header)

- Фон: `#0D0D0D` с `backdrop-filter: blur(12px)` при скролле
- Позиция: fixed top, z-index высокий
- Левая часть: логотип
- Правая часть: иконка корзины (с бейджем), переключатель языка, кнопка «Войти»
- На мобайле: бургер-меню (три линии → крест), меню открывается снизу (bottom sheet)

### Бейдж корзины

- Маленький круг `#C8A96E` с числом `#0D0D0D` поверх иконки корзины

### Тег / Пилюля (Badge)

- Например: «Хит», «Новинка», «Острое»
- Фон: `#C8A96E1A` (золото с прозрачностью 10%)
- Текст: `#C8A96E`
- Рамка: 1px `#C8A96E40`
- Размер: 11px, uppercase, letter-spacing 0.1em

## Иконки

**Библиотека:** Lucide Icons (MIT-лицензия, SVG, легко кастомизировать).  
**Цвет иконок по умолчанию:** `#8A8A8A`  
**Цвет активных/акцентных иконок:** `#C8A96E`  
**Размер:** 20×20px в интерфейсе, 24×24px в крупных блоках.

---

## Изображения

- **Фото блюд:** профессиональные, тёмный или нейтральный фон, съёмка сверху или под углом 45°. Цветовая температура тёплая.
- **Фоновые текстуры:** тонкая текстура тёмного дерева или матового камня (opacity 5–8%) — только в hero-секции.
- **Формат:** WebP с JPEG-фолбэком. Lazy loading для всего кроме hero.

## Анимации и переходы

| Элемент              | Анимация                          | Длительность |
|----------------------|-----------------------------------|--------------|
| Кнопки hover         | color + scale(1.02)               | 200ms ease   |
| Карточки hover       | border-color + translateY(-2px)   | 200ms ease   |
| Страничные переходы  | fade (opacity 0→1)                | 300ms ease   |
| Модальные окна       | fade + scale(0.96→1)              | 250ms ease   |
| Toast-уведомления    | slide-in снизу                    | 250ms ease   |
| Загрузка (skeleton)  | pulse (opacity 0.4→0.8)           | 1.2s infinite|

**Принцип:** анимации быстрые и сдержанные. Никаких слайдеров с задержками. Движение — только там, где помогает понять интерфейс.

## Адаптив (Breakpoints)

| Название   | Ширина      | Устройства              |
|------------|-------------|-------------------------|
| Mobile     | < 640px     | Смартфоны               |
| Tablet     | 640–1024px  | Планшеты                |
| Desktop    | > 1024px    | Ноутбуки, мониторы      |

**Mobile-first:** все стили пишутся от мобайла вверх.  
**Сетка:** CSS Grid + Flexbox. Карточки меню: 1 колонка (mobile) → 2 (tablet) → 3–4 (desktop).  
**Максимальная ширина контента:** 1280px, по центру.

---

## Специфика японской эстетики

- **Пустое пространство (Ma / 間):** щедрые отступы между секциями — минимум 80px вертикально на десктопе. Дышащий макет.
- **Асимметрия:** не все блоки строго по центру — иногда текст слева, иногда смещение для визуального интереса.
- **Тонкие линии:** горизонтальные разделители — 1px, Gold Subtle `#8B6F3E`, opacity 40%.
- **Типографский контраст:** крупные заголовки Cormorant Garamond рядом с мелким DM Sans создают ритм.

---

## Запрещённые паттерны (Anti-patterns)

- Нет ярких цветов кроме золотого акцента
- Нет скруглённых кнопок (border-radius > 6px)
- Нет теней с белым или ярким glow
- Нет стоковых японских иллюстраций (самураи, веера и т.п.)
- Нет анимаций дольше 400ms в интерфейсных действиях
- Нет фонов белее `#1A1A1A` на основных страницах
