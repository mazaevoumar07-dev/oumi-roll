# F-02 — Корзина

**Этап:** Frontend (завершён)

**Назначение:** Клиент собирает список роллов перед оформлением заказа.

---

## Что видит пользователь

- Список добавленных позиций
- Количество каждой позиции
- Цена за единицу и итого по позиции
- Общая сумма заказа
- Кнопка "Оформить заказ"

---

## Действия пользователя

- Увеличить или уменьшить количество позиции
- Удалить позицию из корзины
- Перейти к оформлению заказа

---

## Исключительные случаи

- Корзина пустая → кнопка "Оформить заказ" неактивна
- Пользователь уменьшил количество до 0 → позиция автоматически удаляется из корзины
- Ролл стал "Нет в наличии" пока пользователь смотрел меню → при переходе к оформлению показывается предупреждение "Один или несколько роллов больше недоступны"
