# Отслеживание заказа после оформления

> Связанные функции: F-07 (Отслеживание статуса заказа), F-13 (GPS-отслеживание курьера)

---

## Что происходит после успешной оплаты

1. Stripe подтверждает оплату → сервер создаёт заказ
2. Клиент видит экран подтверждения:
   - Номер заказа
   - Ссылка для отслеживания: `/suivi/[id]?token=[tracking_token]`
   - Таймер на отмену (3 минуты) — см. [03_cancellation_refund.md](03_cancellation_refund.md)
3. Клиент может открыть ссылку в любой момент чтобы увидеть статус

---

## Что видит клиент на странице отслеживания

- Номер заказа
- Текущий статус (обновляется автоматически каждые 10 секунд)
- Время оформления заказа
- Ожидаемое время доставки: **30 минут + 5 минут за каждый км от ресторана**
- Карта с курьером — появляется только когда статус "В доставке"

---

## Статусы заказа

```
Новый → Готовится → В доставке → Выполнен
```

| Статус | Кто меняет | Что видит клиент |
|---|---|---|
| Новый | Создаётся автоматически при оплате | Заказ принят, ожидайте |
| Готовится | Администратор | Ваш заказ готовится |
| В доставке | Администратор | Курьер в пути + карта |
| Выполнен | Администратор | Заказ доставлен |
| Отменён | Клиент (3 мин) или администратор | Заказ отменён |

---

## Карта с курьером

Появляется только когда статус "В доставке".

**Что видит клиент:**
- Маркер курьера на карте (Google Maps)
- Маркер ресторана
- Маркер адреса клиента
- Положение курьера обновляется каждые 10 секунд

**Как это работает:**
- Администратор нажимает "Передать курьеру" → сервер генерирует ссылку для курьера
- Администратор отправляет ссылку курьеру (SMS / WhatsApp)
- Курьер открывает ссылку на телефоне → браузер просит разрешение на GPS → курьер разрешает
- Телефон курьера автоматически передаёт координаты каждые 10 секунд

---

## Ситуации

### Клиент потерял ссылку на отслеживание
- Если клиент зарегистрирован — может войти в аккаунт и увидеть историю заказов (без токена)
- Если не зарегистрирован — без ссылки статус не увидит

### Курьер не открыл ссылку или не дал разрешение на GPS
- Карта не показывается
- Клиент видит только статус "В доставке" без карты

### Курьер потерял интернет
- Последняя известная позиция остаётся на карте
- Обновление прекращается до восстановления связи

### Заказ выполнен
- Карта скрывается
- Статус меняется на "Выполнен"

### Заказ отменён
- Статус показывается как "Отменён"
- Карта не показывается

### Клиент ввёл неверный токен или номер заказа
- Страница возвращает ошибку: "Заказ не найден"
