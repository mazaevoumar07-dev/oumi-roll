## Цель

Закрыть оставшиеся пункты, которые блокируют реальный публичный запуск: домен, регистрацию клиентов и оставшиеся фото меню. Бэкенд (API, БД, интеграции) уже готов — это финальные шаги перед тем, как пустить настоящих клиентов.

> **Сайт пока личный проект, без реальных клиентов.** Пользователь сознательно откладывает всё, что нужно только для публичного запуска — не предлагать эти пункты как срочные, пока сам не поднимет тему:
> - **Stripe Live mode** — решение от 2026-07-05, остаётся в Test mode
> - **Покупка домена** — решение от 2026-07-14, сайт остаётся на `oumi-roll.vercel.app`
> - **Снятие Twilio Trial** — решение от 2026-07-14, см. пункт 2 ниже

## Что нужно сделать

### 1. Домен — отложено (2026-07-14)
Личный проект, домен пока не нужен. Вернуться к этому пункту, когда решится использовать сайт по-настоящему.
- [ ] Купить домен (например, `oumiroll.fr` на [OVH](https://www.ovh.com))
- [ ] Подключить в Vercel → Settings → Domains
- [ ] Прописать DNS-записи у регистратора по инструкции Vercel
- [ ] Дождаться выпуска SSL-сертификата (обычно 5–30 минут)
- [ ] Обновить `NEXT_PUBLIC_APP_URL` в Vercel Production на боевой домен
- [ ] Проверить, что CORS (`next.config.ts`, читает `NEXT_PUBLIC_APP_URL`) не блокирует запросы с нового домена

### 2. Регистрация клиентов (Twilio Trial) — диагноз подтверждён, снятие отложено
См. [known_risks.md](../../docs/spec/known_risks.md) — «Регистрация клиента не работает».
- [x] Проверить в Supabase Dashboard → Authentication → Providers → Phone, что привязаны верные Twilio credentials
- [x] Протестировать регистрацию на сайте с реальным номером телефона — подтверждено 2026-07-14: владелец добавил свой номер в Twilio → Verified Caller IDs, SMS-код дошёл, регистрация прошла полностью
- [ ] Добавить способ оплаты в Twilio-аккаунт, подключённый в Supabase Dashboard — снимает ограничение Trial для ЛЮБЫХ номеров (отложено, не нужно для личного использования)
- [x] Отметить статус в `known_risks.md`

### 3. Фото меню — 4 позиции ещё на заглушках
Проверено через Supabase Storage (2026-07-13) — у всех остальных позиций меню уже реальные фото, кроме:
- [ ] Jus d'orange (Boissons)
- [ ] Jus de pomme (Boissons)
- [ ] Mayo Épicée (Sauces)
- [ ] Dragon Roll (Spécialités)

### 4. Финальная проверка перед публичным запуском
- [ ] Полный тестовый заказ end-to-end (тест-картой, Stripe Test mode)
- [ ] SMS администратору о новом заказе пришло
- [x] Регистрация нового клиента с реальным (верифицированным) номером прошла — подтверждено 2026-07-14
- [ ] Повторно пройти чеклист [deployment_spec.md](../../docs/spec/deployment_spec.md) — пункты про домен
